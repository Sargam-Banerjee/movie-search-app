# Movie Search App
Search for any movie and instantly view results with posters, titles and release years.

Built with: HTML, CSS, Vanilla JavaScript
API: OMDB API (omdbapi.com)

Features:
→ Search any movie by title
→ Responsive grid layout — works on all screen sizes
→ Handles movies with no poster gracefully
→ Error handling for failed searches and API errors
→ Enter key support via HTML form submit

Key concepts: fetch API, async/await, DOM manipulation, CSS Grid (auto-fill + minmax)
